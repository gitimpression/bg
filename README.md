# 校园办公系统（bg）

## 概述

基于`Vue2` 和 `Element-UI`开发

## 依赖版本

+ vue  2.7.14
+ element-ui  2.15.12
+ axios  1.1.3
+ vue-router  3.6.5
+ vuex  3.6.2

## 功能

-   [x] 登录/注销
-   [x] 注册
-   [x] 基本信息
-   [x] 登录记录
-   [x] 修改密码
-   [x] 操作日志
-   [x] 用户中心
-   [x] 办事大厅
-   [x] 权限管理
-   [x] 系统菜单管理
-   [x] 用户搜索
-   [x] 用户日志
