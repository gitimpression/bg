# 校园办公系统（bg）

## 概述

基于`SpringBoot2.3.4` 和 `mybatis2.2.2`开发，使用AOP对用户更新、登录操作进行记录，使用拦截器对权限接口进行拦截.

## 依赖一览

+ SpringBoot  2.3.4
+ mybatis  2.2.2
+ mysql  8.0.29
+ spring-aop
+ spring-web
+ pagehelper 5.3.0
+ lombok
+ jjwt  0.9.1
+ fastjson2  2.0.19

## 功能

-   [x] 登录/注销
-   [x] 注册
-   [x] 基本信息
-   [x] 登录记录
-   [x] 修改密码
-   [x] 操作日志
-   [x] 用户中心
-   [x] 办事大厅
-   [x] 权限管理
-   [x] 系统菜单管理
-   [x] 用户搜索
-   [x] 用户日志
